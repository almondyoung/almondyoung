# Yangtao

I build edge video and AI systems: Go control planes, C++/FFmpeg/WebRTC media pipelines, and Rust tooling.

Current focus:
- low-latency video streaming and surveillance edge agents
- device orchestration and observability for Jetson-class edge nodes
- AI agent infrastructure and practical model-serving systems

Selected public work:
- [vpm](https://github.com/almondyoung/vpm): Go video processing manager with REST APIs, task queues, and worker orchestration.
- [tuya-ipc-video](https://github.com/almondyoung/tuya-ipc-video): TLS TCP service for receiving Tuya IPC video/audio streams with Protobuf packets.
- [zlm-webhook-debug](https://github.com/almondyoung/zlm-webhook-debug): Tiny Go helper for debugging ZLMediaKit webhook payloads.
- [SpecialFFLV](https://github.com/almondyoung/SpecialFFLV): C++ experiments around FLV and low-level media processing.
- [opvs](https://github.com/almondyoung/opvs): Python video processing service experiments for media pipelines.
- [ZLMediaKit](https://github.com/almondyoung/ZLMediaKit): Media-server fork used for WebRTC/RTSP/RTMP exploration.

I am pruning older forks, mirrors, and reading archives so this profile better reflects current work.
